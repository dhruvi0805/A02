Directions on using Webstorm
1) Go to: https://www.jetbrains.com/webstorm/ 
2) Click on download 
3) Create a new account and download the file
4) Open the .exe file 
5) Install the program and select the programming languages you want
6) Once installed, open the Webstorm app and login through your ID 
7) Accept the terms and conditions and the app is ready to use 

Glossary: 
1) Branch: A set of files that changes over time. it is used to isolate development work without affecting other branches in the repository.
2) Clone: Copying a file or making a duplicate. 
3) Commit: saves/ records changes to a file.
4) Fetch: to retrieve new work done by other people 
5) GIT: GIt is a control system that lets developers keep track of their code and manage GIT repositories. 
6) Github: It is a service that lets users host, share, and manage code files on the internet.
7) Merge: Merge combines multiple sequences of commits into one unified history.
8) Merge Conflict: Merge conflicts happens when people edit the same line on the same file or someone deletes a file when someone else edits it. 
9) Push: Push is used to upload local repository content to a remote repository.
10) Pull Remote: Pull remote is a shortcut to git fetch and git merge, which downloads content from a remote repository and immediately update the local repository to match that content
11) Repository: A place to store code, files and revision history
